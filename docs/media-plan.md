# Media Plan

This project needs photos and video because the best parts are physical.

A wiring table can explain the build, but a shot of the original power switch waking the machine up will sell it in two seconds.

## Best video angle

Do not make the video about being a genius.

Make the machine the main character.

The pitch:

> This dead 1985 Apple IIc Monitor became a portable Raspberry Pi Macintosh fever dream with a boot chime selector, Bluetooth ADB keyboard, 3D printed mouse, Apple logo gestures, RetroPie, and a velvet-lined charging bag.

That is enough. The project is already interesting. It does not need chest-thumping.

## Video structure

### 0:00 to 0:30: cold open

Show the finished thing immediately.

Shots:

- dead-looking Apple monitor on desk
- flip original power switch
- boot chime plays
- screen lights up
- Apple logo tap launches app
- four taps opens RetroPie
- controller starts game
- quick shot of keyboard, mouse, bag

Voiceover idea:

```text
This started as a dead Apple IIc monitor from 1985. The CRT was gone, the case was too good to throw away, and I had this dumb idea stuck in my head for years. So I turned it into Macintosh PiForma.
```

### 0:30 to 1:30: what is it?

Explain the premise fast.

```text
It is not a restoration. It is not really a Hackintosh either. It is a Raspberry Pi 4 stuffed into an Apple IIc Monitor shell, dressed up like a Macintosh Performa that fell through a time portal.
```

Show:

- front
- rear ports
- side/profile
- keyboard
- mouse
- bag

### 1:30 to 3:30: show the fun features

Order matters. Start with the stuff normal people can understand.

1. Original power switch
2. Boot chime selector
3. Apple logo touch gestures
4. RetroPie without keyboard/mouse
5. Bluetooth ADB keyboard
6. 3D printed Apple-style mouse
7. Bag with wireless trickle charging

### 3:30 to 7:00: open it up

Show the inside.

Shots:

- shelf assembly
- Raspberry Pi on top shelf
- custom 5V JST rail
- LCD controller on middle shelf
- USB speaker on bottom shelf
- Anker battery wedged in rear
- battery plunger
- YS-M3 chime board and speaker
- source select wiring
- display ribbon taped down

Suggested line:

```text
The inside became a game of millimeters.
```

Another good line:

```text
This 5V rail is not fancy. It is perfboard, JST connectors, and solid-core wire. But it makes the whole thing serviceable.
```

### 7:00 to 9:00: show the ugly parts

People trust builds more when you show the failures.

Include:

- wrong display boards
- too many HDMI ribbons
- noisy Amazon power adapter causing EMI weirdness
- relay/touch source switching abandoned
- fragile LCD ribbon
- battery fit being too tight

Suggested line:

```text
A lot of this project was not genius. It was ordering every possible cable direction because I had no idea which way physics was going to betray me.
```

### 9:00 to 11:00: software world

Show the desktop and apps.

Shots:

- Mac OS 9-ish desktop
- AtEase
- ControlStrip
- About This PiForma
- Clippy
- Weather Channel
- Flying Toasters
- RetroPie
- SheepShaver / Mac OS 9
- Winamp / QMMP

Suggested line:

```text
This is not macOS. It is Raspberry Pi OS wearing a thrift-store Macintosh costume, and honestly that is kind of the point.
```

### 11:00 to end: close

Bring it back to the project feeling finished.

Suggested line:

```text
I do not think this is the most practical computer I own. But it might be the most complete expression of the kind of computer I wanted to build when I was reading Mac mod forums twenty years ago.
```

End with:

- boot chime
- Apple logo four taps
- game launch
- bag zip-up
- GitHub repo on screen

## The tone

Aim for:

```text
This is ridiculous. I love it. Here is how it works.
```

That tone gives people permission to enjoy the project without needing you to prove anything.
