# Repo Finalization Checklist

Before publishing:

- [ ] Add final project photos under `docs/images/`
- [ ] Add STL files under `stl/`
- [ ] Add live Python scripts under `scripts/`
- [ ] Add live systemd unit files under `systemd/`
- [ ] Add live config files under `config/`
- [ ] Add Apple-style mouse shell source link under `references/`
- [ ] Add exact LCD board model text if visible on the board
- [ ] Add exact JST connector family if confirmed
- [ ] Add exact Raspberry Pi OS version from `cat /etc/os-release`
- [ ] Add exact microSD size from `lsblk`
- [ ] Add photos of power rail and rear panel
- [ ] Review safety notes before publishing
